# react-gallery
one photo gallery project based on react

根据慕课网 “React实战-打造画廊应用” 课程，使用新版webpack以及ES6重构

使用Yeoman generator for ReactJS and Webpack创建

## how to use
- clone this repo
- npm install 
- `npm start` or `npm run serve`
